Reproducibilidad y Control de Versiones para las Ciencias Sociales

Julio César Iturra Sanhueza - jciturra@uc.cl

Presentación

El presente documento tiene el objetivo de introducir a investigadores e investigadoras de las Ciencias Sociales al uso de herramientas para el control de versiones para la realización de investigación que permita cumplir con los estándares de Reproducibilidad, Apertura y Transparencia (Nosek et al., 2015)
